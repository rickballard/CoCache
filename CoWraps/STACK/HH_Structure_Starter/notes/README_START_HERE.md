# Hitchhiker Plan (HH)

This folder contains the working documents and autostitched megascroll for the Hitchhiker Plan (HH).

- `sections/` — Individual numbered content sections
- `megascroll.md` — Continuously updated full scroll
- `_toc.md` — Table of contents (auto-updated)
- `_history/` — (Optional) Snapshots of past scrolls

Use `stitch.ps1` (in progress) to recompile the megascroll locally if needed.
