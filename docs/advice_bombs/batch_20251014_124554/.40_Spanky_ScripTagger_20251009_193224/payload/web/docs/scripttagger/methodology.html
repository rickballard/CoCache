Methodology & Disclaimers

What ScripTagger does

  • Segments content into ~20-sentence windows.
  • Tags five dimensions: TruthAlignment, EvidenceHygiene, CivicConduct, Anti-Manipulation, AmplificationQuality.
  • Builds a two-panel summary.
  • Transcript optional & off by default.

Noisy venues & mic profiles

Choose Auto / Crowd / Stage / Quiet. Optional 3s noise calibration; system privacy indicators visible.

Preliminary → Upgraded transcripts

Live captures can be noisy. When official media arrives, re-align & mark provenance; summaries update.

Limitations

  • Transcripts & identity inference can be wrong.
  • Crowd agreement is a signal, not truth.
  • Not legal/medical advice.

Privacy

Local-first; opt-in to contribute signed JSON to an append-only log.
